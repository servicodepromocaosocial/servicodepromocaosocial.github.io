    Política de Privacidade - LGPD Última atualização: 14/07/2025

    Este site, hospedado no GitHub Pages, respeita a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018) e está comprometido com a transparência no tratamento de dados pessoais.

    1. Dados Coletados Dados de Navegação: Podemos coletar informações como IP, tipo de navegador, páginas acessadas e tempo de visita (via cookies ou ferramentas como Google Analytics).

    Dados Fornecidos por Você: Se você preencher formulários (ex.: newsletter), coletaremos nome, e-mail e outras informações fornecidas.

    1. Finalidade do Tratamento Seus dados podem ser usados para:

    Melhorar a experiência no site.

    Enviar comunicações (se consentido).

    Gerar estatísticas de acesso (anonimizadas quando possível).

    1. Bases Legais (LGPD) O tratamento é baseado em:

    Consentimento (para cookies e newsletters).

    Legítimo interesse (para análise de tráfego).

    1. Cookies e Tecnologias Similares Utilizamos cookies para funcionalidade e analytics. Você pode gerenciá-los no popup de consentimento ou nas configurações do navegador.

    2. Compartilhamento de Dados GitHub Pages: Como o site é estático, o GitHub não coleta dados além dos logs padrão (veja GitHub Privacy Policy).

    Ferramentas de Terceiros: Serviços como Google Analytics podem processar dados (sujeitos às suas políticas).

    1. Seus Direitos (LGPD) Você pode:

    Acessar, corrigir ou excluir seus dados.

    Revogar consentimento (via popup ou contato).

    Solicitar portabilidade ou oposição ao tratamento.

    Contato para DPO (Encarregado de Dados): [Incluir e-mail ou formulário de contato, se aplicável].

    1. Medidas de Segurança Adotamos medidas técnicas para proteger seus dados, mas lembre-se de que o GitHub Pages não é um ambiente com recursos avançados de segurança (evite coletar dados sensíveis).

    2. Alterações na Política Esta política pode ser atualizada. A versão mais recente estará disponível nesta página.
